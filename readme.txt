== Description ==

Smart Contracts for Web3 plugin allows you to easily create Smart Contracts that can be deployed on any Network using Metamask. 
Create DApps through Smart Contracts and take advantage of Blockchain security.

== Installation ==

= From WordPress.org =
1. Download miniOrange Web3 Smart Contracts plugin.
2. Unzip and upload the `Web3 Smart Contracts` directory to your `/wp-content/plugins/` directory.
3. Activate Web3 Smart Contracts from your Plugins page.

= From your WordPress dashboard =
1. Visit `Plugins > Add New`.
2. Search for `Web3 Smart Contracts`. Find and Install `Web3 Smart Contracts`.
3. Activate the plugin from your Plugins page.
